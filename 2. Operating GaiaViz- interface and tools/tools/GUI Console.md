### (GUI) Command Console

| Hotkeys     | Functions                                                   |
| ----------- | ----------------------------------------------------------- |
| Enter       | Enters Command Console (from Game Mode).                    |
| `?` + Enter | Display the command usage help                              |
| Esc         | leave Command Console and return keyboard to **Game Mode**. |

Use this console to customize your selection.
